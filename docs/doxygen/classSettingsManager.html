Detailed Description

A thread-safe Singleton manager for storing and retrieving configuration settings.

  • This class uses a std::map coupled with a std::variant (SettingValue) to store mixed-type configuration variables associated with string keys. It utilizes std::mutex to guarantee thread-safe read and write operations across the application.
