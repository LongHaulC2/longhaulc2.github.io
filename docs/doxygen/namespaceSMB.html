DWORD SMB::read_pipe_dynamic ( HANDLE  h_pipe,
std::vector< uint8_t > &  out_buffer 
)

Dynamically reads data from a specified named pipe into a buffer.

  • Reads in chunks of 8192 bytes, handling cases where the incoming data is larger than the chunk size (ERROR_MORE_DATA) by looping until the transfer completes.
  • Parameters
    h_pipeA handle to the pipe to read from.
    out_bufferA reference to a byte vector where the read data will be stored. This buffer is cleared before reading begins.
    Returns
    DWORD Returns ERROR_SUCCESS (0) on success, or a Windows API error code on failure.
