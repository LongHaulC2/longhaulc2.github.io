Detailed Description

Encapsulates wrapped Windows API functions to manage resolution and prevent naming collisions.

The standard Windows API functions within this namespace are direct 1:1 wrappers. For parameter and return type details on these functions, refer to the official Microsoft documentation.

void WinApi::EnsureModuleLoaded ( LPCSTR  moduleName)
inline

Ensures a specified module is loaded into memory.

  • Prevents lazy_importer from crashing by checking if a module is already in the PEB, and calling LoadLibraryA if it is not found.
  • Parameters
    moduleNameThe name of the module (DLL) to verify and potentially load.
